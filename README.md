# 🤬 JakBite Bot

> **Пасивно-агресивний таск-менеджер у Telegram, який змусить тебе працювати не через мотивацію, а через сором.**

Досить ніжних трекерів, які надсилають милі сповіщення на кшталт *"Будь ласка, знайдіть час на спорт"* 💖. `JakBite Bot` — це цифровий наглядач, який нагадає про дедлайн так, що тобі захочеться видалити Telegram, але ти підеш і виконаєш таску.

---

## 🔥 Що бот вміє вже зараз (Core Features)

Проєкт активно розвивається і вже має потужний арсенал для боротьби з твоєю лінню:

- 🎭 **Три режими наглядача:** Обирай, як саме бот має з тебе знущатися:
  - `Нянька 🌸` — лагідні нагадування для слабких духом.
  - `Кат 😈` — класичні пасивно-агресивні стусани та сарказм.
  - `УЛЬТРА-САДИСТ 💀` — капс, образи, стікери та тотальне цифрове знищення.
- 📊 **Дошка Ганьби (Профіль):** Бот веде твою статистику. Він вираховує твій **"Індекс ліні"** у відсотках і видає відповідні звання (від *Амеби* до *Термінатора дедлайнів*).
- 🚨 **Спам-режим:** Можливість встановити нагадування "кожні 5 хвилин", щоб бот не дав тобі спокою, поки ти не закриєш завдання.
- 💾 **Повноцінна база даних:** Усі твої завдання, борги та статистика надійно зберігаються в `SQLite`.
- 🖼 **Інтерактивний UI:** Зручні inline-клавіатури, календар для вибору дати та динамічні стікери, що супроводжують кожне нагадування.

## 🚀 У найближчих планах (Roadmap)

Ми не збираємося зупинятися на досягнутому. Попереду ще більше болю для прокрастинаторів:
- [ ] **Капча для лінтяїв:** Щоб відкласти таску (кнопка `Snooze`), доведеться вручну написати принизливу фразу (наприклад, *"Я лінива амеба"*).
- [ ] **Ранкові та вечірні дайджести:** Розсилка списку боргів на день та жорстке підбиття підсумків ввечері.
- [ ] **Система "Стріків" (Streaks):** Трекінг щоденної активності зі згорянням прогресу в разі пропуску.

## 🛠 Технологічний стек

- **Language:** Python 3.10+
- **Framework:** [aiogram 3.x](https://github.com/aiogram/aiogram) (Asynchronous Telegram Bot API)
- **Scheduler:** [APScheduler](https://github.com/agronholm/apscheduler) (для фонових дедлайнів та спам-атак)
- **Database:** SQLite (вбудована)
- **UI Components:** `aiogram_calendar` для зручного вибору дат.
